# customblock

这是一个iOS端Safari的扩展。<br/>
属于自娱自乐的个人项目。<br/>
安装后可以去掉一些恼人的强制跳转app的提示或者脚本。<br/>
我会慢慢加上一些个人有可能会访问到的网站<br/>
会去掉一些广告、去掉下载app提示<br/>
或者让手机端Safari更好的访问这个网站<br/>
也欢迎提issues，我会抽空完善<br/>

暂时不打算上架，<br/>
但是可以通过testfight下载：<br/>
https://testflight.apple.com/join/bQyq38by


### 目前支持的网站如下
- baidu
   - 首页去掉了新闻
   - 搜索结果页去掉了部分广告
   - 百家号去掉了提示安装app的弹窗
- 3g.163.com
   - 首页新闻列表去掉了跳转app的方式
   - 去掉了部分广告
- zhihu
   - 文章默认展开
   - 去掉打开app的弹窗
- 36kr
- sina
